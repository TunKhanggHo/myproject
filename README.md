# Bài Tập Lớn - Phân Tích Thiết Kế Thuật Toán

Kho lưu trữ này chứa các bài toán thuật toán được triển khai bằng ngôn ngữ **Python**.

## 📌 Danh sách các bài toán
* **1_BaiToanSanXuat.py**: Thuật toán giải bài toán tối ưu hóa sản xuất.
* **2_PhanCongCongViec.py**: Bài toán phân công công việc bằng phương pháp nhánh cận / tham lam.
* **3_CaiTui.py**: Thuật toán Quy hoạch động giải bài toán Cái túi (Knapsack).
* **4_VanChuyenXang.py**: Bài toán tối ưu hóa vận chuyển.

## 🛠️ Công nghệ sử dụng
* Ngôn ngữ: Python 3
